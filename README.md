
# Smart E-Bill System

Automated billing system developed using Python and SQLite to reduce manual billing effort and improve transaction processing efficiency.

## Features
- Automated bill generation
- Stores transaction data using SQLite database
- Improves billing speed and accuracy
- User-friendly interface

## Technologies Used
- Python
- SQLite
- Android Studio
